# 📂 Flickson's WhatsParsie

**Flickson's WhatsParsie** is a lightweight, privacy-first tool designed to view exported WhatsApp chat archives. It runs entirely in your browser, allowing you to load and read multiple `.zip` exports without ever uploading your data to a server.

---

### 🚀 Status: In Development
This project is currently a work-in-progress. The goal is to create a seamless, offline experience that mirrors the WhatsApp Web interface.

### ✨ Key Features (Planned & Current)
* **Local-First:** No database, no servers. All parsing happens in your browser's RAM.
* **Multi-Chat Support:** Load an entire folder of ZIP exports at once.
* **Privacy by Design:** Your messages never leave your local machine.
* **Portable:** A single-file solution (HTML/React) for easy use across different devices.

### 🛠️ Built With
* **React** & **Tailwind CSS** for the UI.
* **JSZip** for handling compressed exports.
* **Regular Expressions** for parsing chat logs.
